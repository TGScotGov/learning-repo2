# Method 

When updating anything file and pushing up to Github always remember these FOUR steps

## Process
1. Save the file in Visual Studio Code, otherwise nothing updates
2. Git add . (adds everything)
3. git commit -m "make a comment"
4. git push, this commits the change up to Github from the local directory
